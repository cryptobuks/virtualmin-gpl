Délai minimum entre les réponses automatiques
Quand reponses automatiques sont activées, cette option peut être utilisée pour limiter la vitesse à laquelle les réponses sont envoyées à chaque expéditeur. Si un second message arrive du même expéditeur dans la période indiquée ici, une réponse automatique ne sera pas envoyé. Ceci est utile pour prévenir les boucles de courrier dans lequel deux répondeurs automatiques envoient des e-mails uns les autres!

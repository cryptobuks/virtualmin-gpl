Action pour d'autres expéditeurs
Ce menu détermine la façon dont les serveurs de messagerie traitent courriel qui semble provenir de ce domaine, mais pas à partir d'un serveur permis spécifié en utilisant les champs ci-dessus. Si vous êtes absolument certain que tout le courrier pour le domaine passe par cet hôte de serveur virtuel ou d'une autre sur la liste autorisée, vous devriez sélectionner Rejeter.

Destinations pour les spams
Ce champ détermine ce qui se passe aux e-mails qui ont été identifiés par SpamAssassin comme spam, en fonction de leur note calculée et le seuil de spam du destinataire. Les options disponibles sont :

Jeter
Tous les messages classés comme spam seront immédiatement supprimés.
Livrer normalement
Spam sera livré à la boîte aux lettres du destinataire, mais avec en-têtes X-Spam qui peuvent être utilisées pour filtrer par un client de messagerie.
Écrire dans le fichier de spam standard ~/mail/spam
Spam est écrit dans le dossier standard pour l'analyse manuelle et suppression.
Écrire dans le fichier de courrier sous le répertoire d'accueil
Spam est enregistré dans le fichier courrier sous le répertoire personnel de l'utilisateur que vous entrez. Pour écrire à un répertoire Maildir, mettre un / à la fin du nom de fichier.
Transférer à l'adresse e-mail
Tous les messages classés comme spam seront transmis à l'adresse indiquée.
Ecrire à autre fichier
Spam sera écrit dans le fichier de messagerie donné (ou un répertoire si le chemin se termine par /), qui peut être n'importe où sur le serveur.

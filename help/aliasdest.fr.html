Destinations de Alias
Ce tableau de champs détermine ce qui se passe à email envoyé à cet alias. Si plusieurs destinataires sont entrés, le courrier sera envoyé à chacun d'eux. Chaque destination possède un type (choisi à partir du menu), qui peut être l'un des suivants :
  • Adresse mail
    E-mail sera envoyé à l'adresse indiquée dans le champ adjacent.

  • Adresses dans le fichier
    E-mail sera envoyé à toutes les adresses figurant dans le fichier texte dont le chemin est entré dans le champ adjacent.

  • Écrire dans le fichier
    Envoyer à l'alias sera jointe dans le fichier spécifié. Tous les en-têtes et le corps de la non-décodé brut seront inclus.

  • Fournir au programme
    Le programme est entré dans le champ adjacent sera exécuté et votre email à l'alias qui lui est passé en entrée.

  • Réponse automatique à partir du fichier
    Quand email est reçu par cet alias, et le message de réponse automatique sera envoyée à l'expéditeur. Le texte de cette réponse automatique sera prise à partir du fichier entré dans le champ de texte adjacente.

  • Appliquer le fichier filtre
    Courriel envoyé à l'alias sera traitée selon les règles dans le fichier de filtre spécifié, ce qui peut l'amener à être transmis à d'autres adresses ou des fichiers en fonction du contenu réel du message.

  • Boîte de courrier de l'utilisateur
    Email sera livré directement à la boîte aux lettres de l'utilisateur est entré dans la zone de texte adjacente. Cette contourne toutes les adresses d'expédition que l'utilisateur a mis en place. Cependant, vous devez entrer le nom d'utilisateur POP3 plein connexion, comme joe.foo au lieu de simplement joe.

Si vous choisissez les types de destination Adresses dans le fichier, Réponse automatique à partir du fichier ou Appliquer le fichier filtre, lorsque vous modifiez les liens d'alias marquées, Éditer.. apparaîtra à côté de ces destinations pour modifier le contenu du fichier spécifié.

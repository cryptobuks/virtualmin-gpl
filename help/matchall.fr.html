Site correspond à tous les sous-domaines?
Lorsque cette option est activée, les demandes pour toutes les URL dans le cadre du domaine (comme http://abc.exemple.com) afficheront le site Web du serveur virtuel. Cela peut être utile si vous hébergez une application qui peut retourner un contenu différent en fonction du nom de domaine demandé, comme Wordpress MU.

Si le domaine DNS du serveur virtuel est hébergé sur ce système, il va ajouter aussi le joker * pour Enregistrement DNS si Oui est sélectionné.

Admis expéditeur de domaines de messagerie
Ce champ énumère les noms de domaine dont les serveurs de messagerie (identifiés par les enregistrements DNS MX) sont autorisés à envoyer des courriels à partir de ce domaine. Parce que les serveurs de messagerie d'un domaine changent souvent, il est recommandé que vous entrez les noms de domaine ici plutôt que les noms d'hôte du serveur de messagerie dans le champ Les noms d'hôtes d'expéditeurs autorisés, si est approprié.

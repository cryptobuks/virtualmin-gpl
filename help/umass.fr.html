Créer plusieurs utilisateurs
Cette page vous permet de créer plusieurs mail ou FTP utilisateurs à la fois à partir d'un fichier texte. Ce fichier doit indiquer un utilisateur par ligne, et chaque ligne doit avoir le format suivant :

username:realname:password:ftp-flag:email-flag:quota:[extras]:[forwards]:[databases]

Les significations des champs sont :

username
Nom court d'identification de l'utilisateur, tels que joe. Si nécessaire, le préfixe de domaine sera annexé à faire le vrai nom de connexion.
realname
Vrai nom de l'utilisateur, comme Joe Smith.
password
Le mot de passe pour l'utilisateur, en texte brut.
ftp-flag
Si ce paramètre est réglé à 1, l'utilisateur sera autorisé à se connecter via FTP. La valeur de 0 ou laisser vide empêche l'accès FTP. Vous pouvez également utiliser le chemin complet vers un shell valide pour les utilisateurs de boîte aux lettres, comme /usr/bin/scponly.
email-flag
Si ce paramètre est réglé à 1, adresse e-mail principale de l'utilisateur est activée.
quota
Le quota de l'utilisateur de messagerie et le répertoire home, en blocs de 1 Ko de disque.
extras
Une liste séparée par des virgules des adresses e-mail supplémentaires pour cet utilisateur. Ceux-ci doivent être dans des domaines Virtualmin que vous gérez.
forwards
Une liste séparée par des virgules des destinations e-mail d'expédition pour cet utilisateur (adresse généralement comme joe@autredomaine.com).
databases
Une liste des bases de données séparées par des virgules pour accorder l'accès à cet utilisateur. Chacun doit être un type de base de données (mysql ou postgres) et un nom de base de données, séparés par un espace.

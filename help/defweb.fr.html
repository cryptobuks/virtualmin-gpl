Site web par défaut pour l'adresse IP?
Lorsque ce champ est réglé sur Oui, le site Web de ce serveur virtuel sera servi par Apache lorsqu'il reçoit une demande qui ne correspond pas à aucun autre serveur virtuel sur le système. Cela se produit généralement si un utilisateur utilise une URL avec une adresse IP en elle, ou un nom d'hôte qui résout à votre système, mais ne correspond pas à un domaine Virtualmin.

Autoriser l'accès aux bases de données
Ce champ vous permet de sélectionner les bases de données appartenant par ce serveur virtuel auxquelles l'utilisateur peut également se connecter. Pour ajouter une base de données, sélectionnez-le dans la liste de gauche et cliquez sur la flèche -> .

Login et mot de passe de l'utilisateur seront les mêmes que celles qu'il utiliserait pour FTP ou IMAP. Par défaut, il sera accordé un accès complet à toutes les tables de bases de données sélectionnées.

MySQL supporte noms d'utilisateur jusqu'à 16 caractères, donc pour un utilisateur avec un nom long de compte seulement les 16 premiers caractères sont nécessaires lorsque l'utilisateur se connecte à la base de données.

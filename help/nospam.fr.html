Vérifiez email pour spam et les virus?
Par défaut, lorsque le spam et les virus filtrage est activée pour un serveur virtuel, il sera applicable à tous les boîtes aux lettres dans ce domaine. Cependant, vous pouvez désactiver ce pour un certain compte, comme celui qui est configuré pour faire du filtrage de spam sur le client, ou est le destinataire du spam pour le domaine. Pour ce faire, modifiez cette option à Non.

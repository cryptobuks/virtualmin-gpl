Créer une base de données MySQL?
Si elle est sélectionnée, un compte MySQL sera créé pour ce domaine, généralement avec le même nom que la connexion administrative du domaine. Si vous utilisez les paramètres du modèle par défaut, une base de données MySQL sera créé pour le serveur virtuel. Le nom de la base sera la première partie du nom de domaine - donc si vous créez le domaine votredomaine.com, la base de données serait appelé votredomaine.

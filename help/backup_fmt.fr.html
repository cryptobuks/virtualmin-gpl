Format de sauvegarde
Cette option contrôle le format de fichier que Virtualmin va utiliser pour cette sauvegarde. Les options disponibles sont:
Fichier d'archivage unique
Tous les serveurs virtuels stockent dans un fichier unique, à l'emplacement indiqué par le Destination de Sauvegarde.
Un fichier par serveur (ancien format)
Chaque serveur virtuel est sauvegardé dans son propre fichier séparé, dans le répertoire spécifié par la Destination de Sauvegarde.
Un fichier par serveur, dans le nouveau format
Chaque serveur virtuel est sauvegardé dans son propre fichier séparé, mais dans un format différent de l'option précédente. Ce nouveau format est supérieur car il ne nécessite pas un grand fichier créé dans le répertoire /tmp première, ce qui économise de l'espace disque.
La dernière option est recommandée, car elle a permis serveurs virtuels individuels à être plus facilement restaurées, et économise de l'espace lors de la sauvegarde sur disque temporaire.

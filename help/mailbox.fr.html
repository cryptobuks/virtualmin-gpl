Adresse de courrier primaire activée ?
Lorsque cette option est réglée sur Oui, l'utilisateur aura l'adresse e-mail dans le domaine indiqué dans le champ Nom d'utilisateur. Vous pouvez sélectionner Non plutôt si l'utilisateur est en cours de création pour l'accès FTP seulement, ou si vous avez entré une adresse email différente dans le champ Adresses de courrier supplémentaires.

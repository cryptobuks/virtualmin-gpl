Redirections De Site Web
Une redirection fait correspondre certains chemin d'URL comme /foo soit à une adresse URL différente, ou un répertoire différent sur le même serveur virtuel. Cela peut être utilisé pour fournir des chemins d'URL plus amicales sur votre site web, ou pour faire face à la circulation des pages Web vers de nouveaux emplacements.

Cette page répertorie toutes les redirections existantes, et vous permet de créer un nouveau en cliquant sur le lien Ajouter une nouvelle redirection de site Web. Pour modifier une redirection existante, il suffit de cliquer sur son chemin de l'URL source. Pour supprimer plusieurs redirections, cochez les cases à côté d'eux et cliquez sur le bouton Effacer au bas de la page.
